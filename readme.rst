###################
Todo List Application
###################

To do list Application help to you simply add,edit and remove to your to do lists.


*******************
Server Requirements
*******************

PHP version 5.6 or newer is recommended.

It tested with php 7.4.3

************
Installation
************

simply go to Application/config/app-config.php File and edit the file.
then impor sql file. (sql file in the base.)
